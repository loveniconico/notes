-  两个变量中的值进行互换的方法  
    1. 
    ```
        tmp = a;
        a   = b;
        b   = tmp;
    ```
    2. 
    ```
    a = a + b;  
    b = a - b;
    a = a - b;
    ```
    3.
    ```
    a = a ^ b;  // a ^ b ^ b = a
    b = a ^ b;  // b = a ^ b ^ b = a
    a = a ^ b;  // a = a ^ b ^ a = b ^ a ^ a = b
    //这种方法的思想是：利用异或的原理。 a ^ 0 = a, b ^ b = 0 ,则 a ^ b ^ b = a
    ```
